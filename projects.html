MyMusicList (Web)

MyMusicList (web) is a basic web app where users can rate songs and share their review with others. Basic features include account login and signup, song rating and sharing, notifications, simple front-end authentication with JSON Web Tokens, and route protection. I learned so much about the MVC architecture, React, states, components, async calls, and PHP through trial and error. Even though many shortcuts were taken to meet deadlines, I've taken what I learned in this project into future ones!

React, Bootstrap, PHP, MVC, REST API

April, May 2023

Emotion Classification Using ResNet-34

The goal of this project was to train a neural network to detect emotions in human speech, which we implemented using transfer learning. We chose ResNet-34 as the pre-trained convolutional neural network. Our model was able to accurately detect the intended emotion when the emotion was firmly or strongly expressed, but was less accurate when the emotion was expressed more subtly. We extended the project to allow users to record their voice and have the model classify the emotion in their speech in real time.

CNN, ResNet-34, Transfer Learning, TensorFlow, Librosa, Fast.ai (V1)
Github
