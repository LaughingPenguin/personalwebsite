Spring 2023

Improving memory usage and computation efficiency in a network simulator.

This research focused on fixing memory leaks in our network simulator. It was also the first time that I got a chance to look at the codebase! I used logger objects to diagnose where the memory leak is. Afterwards, I tried various methods to fix the memory leak such using different versions of Python/TensorFlow, running the simulation inside specific environments, representing data differently (panda dataframe v. tensors v. numpy matrices), executing garbage collection, etc. Ultimately, I resolved the memory leak by training the model inside a child process, writing the results to a data file, and then killing the child process after completion.

TensorFlow, Pandas, Multiprocessing, Python
